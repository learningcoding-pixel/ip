# "Cat" Chatbot Command List

## Add Tasks
```
<t> | <todo>
<d> | <deadline> | <2/12/2019 1800>
<e> | <event> | <1/12/2019 1800> | <2/12/2019 1800>
```

## List Tasks
```
<list>
```

## Find Tasks
```
<find> | <key>
```

## Modify Tasks
```
<delete> | <index>
<mark> | <index>
<unmark> | <index>
```

## Exit
```
<bye>
```

## Example
```
t | Worship Cat
d | Feed Cat | 1/12/2019 1400
e | Play with Cat | 1/12/2019 1800 | 2/12/2019 2200
list
mark | 2
find | Feed
delete | 3
bye
```
